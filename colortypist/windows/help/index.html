With Colortypist you can type in a color by its name or code, and see it fill your entire screen, in real-time, while you type. You can easily set any color you like as your Desktop Wallpaper, and get a more modern looking Desktop and a faster PC. You can also save any color to a tiny and fully portable HTML file, that can be opened by any browser, PC, tablet or smartphone.

Colortypist can also copy colors for reuse in other applications, and generate sample images of any size filled with the color of your choice. The Random menu lets you easily create beautiful and rare random colors, and the View menu helps you easily convert between HTML color codes, RGB triplets, or color names.

Colortypist has a clean and easy to use interface, and requires no installation: simply download and run.

Last but not least, Colortypist may help you have fun playing with colors, discovering and naming new colors, and even checking the color of your birthday (#800522 for May 22, 1980).

Color Menu

Set As Desktop Wallpaper

Use this menu item to set the current color as a solid color Desktop Wallpaper. Colortypist will first display a confirmation dialog box, asking you to confirm this action.

Save Color As Html...

Use this menu item to save the current color to a tiny, fully portable HTML file. You can open and view the saved color in any browser, on various operating systems and platforms, on PCs, notebooks, tablets, or smartphones.

Copy Color Code

Use this menu item to copy the current color code or name, from the color typing text box to the Windows Clipboard.

Copy Color Sample

Use this menu item to copy a sample image filled with the current color to the Windows Clipboard. The size of the image can be changed by using the Color Sample Size menu item.

Save Color Sample...

Use this menu item to save a sample image filled with the current color. The size of the image can be changed by using the Color Sample Size menu item.

Color Sample Size

Use this menu item to change the size of the sample images that are copied or saved. Simply enter a new width and height in the menu text box, and press Enter. Colortypist can generate sample images of any size, including small 100x100 iconic images, or Full HD, 1920x1080, wallpaper size images.

View Menu

As HTML Hex

Use this menu item to covert the current color code to a HTML hexadecimal color code (e.g. #E303A3).

As RGB

Use this menu item to covert the current color code to a RGB triplet color code (e.g. 227,3,163).

As Integer

Use this menu item to covert the current color code to an integer (numeric) color code (e.g. -1899613).

As Name

Use this menu item to covert the current color code to a color name (e.g. Pink). This option is not available for colors that do not have a standard name.

On COLOURlovers

Use this menu item to open the current color on the COLOURlovers site. COLOURlovers is a creative community where people from around the world create and share colors, palettes and patterns.

Help Menu

Use the Help menu items to open the Colortypist home page, Online Help page (this page), and Share page, and get information about the version number of the app.

Random Menu Item

Use this menu item to generate and fill your screen with a random color.
